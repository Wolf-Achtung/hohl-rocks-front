## Säule 1: hohl.rocks - Deine persönliche Marke

### Neuer Fokus: "Wolf Hohl zeigt, was KI kann"

**Zielgruppe:**
- Neugierige Menschen
- Potentielle B2B-Kunden (Einstieg)
- Menschen die KI verstehen wollen

### Empfohlene Struktur

```
hohl.rocks/
│
├── STARTSEITE (Hero)
│   ├── Video-Hintergrund (behalten - ist gut!)
│   ├── "Wolf Hohl - TÜV-zertifizierter KI-Manager"
│   ├── Tagline: "Ich zeige dir, was KI wirklich kann"
│   └── 3 CTAs: Projekte | KI ausprobieren | Für Unternehmen
│
├── PROJEKTE (NEU - zentral!)
│   ├── ki-sicherheit.jetzt → "KI für Unternehmen"
│   ├── art-radar.berlin → "KI für Kulturinteressierte"
│   ├── akut.jetzt → "KI für Notfälle"
│   └── achtung.jetzt → "KI für sichere Kommunikation"
│   (Jedes Projekt mit: Screenshot, Kurzbeschreibung, Link)
│
├── KI AUSPROBIEREN (interaktiv - dein USP!)
│   ├── Model Battle Arena (behalten - einzigartig!)
│   ├── Prompt-Helfer (vereinfacht)
│   └── "Was kann KI für dich tun?" Quiz
│
├── ÜBER WOLF
│   ├── Vom Kino-Trailer zur KI
│   ├── TÜV-Zertifizierung
│   ├── 25 Jahre Medien-Erfahrung
│   └── "Warum ich KI liebe"
│
└── KONTAKT
    └── Link zu ki-sicherheit.jetzt für B2B
```

### Was ändern bei hohl.rocks?

| Aktuell | Neu |
|---------|-----|
| Zu technisch (Prompt Library, Daily Challenge) | User-freundliche KI-Demos |
| Keine Projekte sichtbar | Projekt-Showcase prominent |
| Unklar wer du bist | TÜV-Zertifizierung + Story |
| Viele technische Features | Weniger, aber beeindruckender |

| Feature | Empfehlung | Begründung |
|---------|------------|------------|
| Video-Hintergrund | BEHALTEN | Macht Eindruck |
| Model Battle | BEHALTEN | Einzigartig, zeigt KI-Vergleich |
| Prompt Library | VEREINFACHEN | Zu technisch für Laien |
| Daily Challenge | ENTFERNEN | Zu nischig |
| Tips-Seiten | ENTFERNEN | Kaum Inhalt, zu technisch |

### Neue Features vorschlagen

| Feature | Beschreibung | Wow-Faktor |
|---------|--------------|------------|

**Zielgruppe:**
- Geschäftsführer, IT-Leiter
- KMUs die KI integrieren wollen

```

│   ├── Integration & Implementierung
│   ├── Schulungen & Workshops
│   └── Fördermittel-Beratung
│
├── ÜBER WOLF HOHL
│   ├── TÜV-Zertifizierung
│   ├── 25 Jahre Medien-Branche
│   ├── Referenzen/Projekte
│   └── Link zu hohl.rocks ("Mehr über mich")
│
└── KONTAKT
    ├── Erstgespräch buchen
    └── Kontaktformular
```

### Readiness-Report Monetarisierung

**Option A: Direkt an Unternehmen**
```
Basis-Report:     Kostenlos (Lead-Generierung)
Standard-Report:  149-299€ (detaillierte Analyse)
Premium-Report:   499-999€ (inkl. Beratungsgespräch)
```

**Option B: White-Label für Partner**
```
Steuerberater, Unternehmensberater, IT-Dienstleister
→ Verkaufen Report unter eigener Marke
→ Du bekommst Lizenzgebühr pro Report
→ Skalierbar ohne deine Zeit
```

**Empfehlung:** Beide Optionen parallel
- Direkt für schnelle Einnahmen
- White-Label für Skalierung

## Die 4 Projekte - Positionierung

### 1. art-radar.berlin
**Pitch:** "KI kuratiert Berlins Kunstszene"
- Zielgruppe: Kunstinteressierte in Berlin
- USP: Automatisch aktualisiert via APIs
- Auf hohl.rocks zeigen als: "KI für Kultur"

### 2. akut.jetzt
**Pitch:** "Dein KI-Notfall-Assistent"
- Zielgruppe: Alle Smartphone-Nutzer
- USP: Funktioniert auch offline, versteckter Notruf
- Auf hohl.rocks zeigen als: "KI rettet Leben"
- **Potenzial:** Sehr hoch! Kooperationen mit Krankenkassen, Behörden?

### 3. achtung.jetzt
**Pitch:** "Bevor du sendest - KI checkt"
- Zielgruppe: Alle die E-Mails schreiben
- USP: Erkennt sensible Daten vor Versand
- Auf hohl.rocks zeigen als: "KI schützt deine Daten"
- **Potenzial:** B2B-Produkt für Unternehmen!

### 4. Readiness-Report (ki-sicherheit.jetzt)
**Pitch:** "Dein KI-Fahrplan in 10 Minuten"
- Zielgruppe: KMU-Geschäftsführer
- USP: Konkreter Plan + Fördermittel
- **Potenzial:** Dein Hauptprodukt für B2B
